                                       News App - iOS

## Features

- Swift with UIKit
- [Snapkit](https://github.com/SnapKit/SnapKit)
- [Alamofire](https://github.com/Alamofire/Alamofire)
- [KingFisher](https://github.com/onevcat/Kingfisher)
- Generic Network Layer
- Localization
- UserDefaults
- Safari Services

## Architech

- MVVM.

## API

- News [API](https://newsapi.org/).
